# Credit Card Data Analysis - Python & Jupyter Notebook

## Project Overview
This project performs end-to-end data analysis on 
credit card data using Python and Jupyter Notebook.

## Tools Used
- Python - Data analysis and programming
- Jupyter Notebook - Interactive coding environment
- Pandas - Data manipulation
- Matplotlib/Seaborn - Data visualization

## Project Structure
- Credit_Card_Analysis.ipynb - Main analysis notebook
- Credit Card Data.xlsx - Raw credit card data
- Customer Acquisition.xlsx - Customer data
- Repayment.xlsx - Repayment data
- Spend.xlsx - Spending data
- Credit Card Case Study 2.pdf - Case study document

## Skills Demonstrated
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Python Programming
- Jupyter Notebook
- Pandas & Matplotlib
